# Developer Preview
Simple application that allows real time view and reload for web applications during development.

# Stack
 - ElectronJS
    - Window and tray manager
 - SASS
    - CSS with superpowers
 - Electron Reload
    - Live reload support at application level
 - Electron Builder
    - Multiple Target
